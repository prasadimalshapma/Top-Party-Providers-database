# Top-Party-Providers-database
Oracle SQL database system for Party booking management
------------------------------------------------------
Oveview:
This project was developed for the 5CI022 Database module at the University of Wolverhampton.TOP PARTY PROVIDERS are a small event entertainment business specializing in clown performances primarily for children's parties. A clown is provided fun activities like balloon art, magic tricks, gifts, and interactive entertainment. The company deals with party bookings, party packages , performers and various party items used at events. 

The type of information that is stored in this database is critical for the daily running of the business and this is because it assists the business in ensuring that there is smooth event organization, management of items, accurate billing and organization. The information is also capable of providing useful reports such as revenue generated per month.

The database supports bookings, packages, performers, items, and payments.

Fearures:
-customer management
-Party packages and performers
-Booking system with items and performers
-Payment tracking
-Referential integrity with PK-FK constraints

Entities:
-Customer
-Package
-Performer
-PartyItem
-Booking
-BookingItem
-Payment
-BookingPerformer
